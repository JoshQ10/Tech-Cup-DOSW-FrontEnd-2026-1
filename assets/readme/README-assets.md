# Guia rapida de assets para README

Guarda aqui las imagenes usadas por `README.md`.

## Estructura sugerida

```text
assets/readme/
  branding/
  mockups/
  flows/
  components/
```

## Reglas de nombres

- 

## Formatos recomendados

- `SVG` para logos e iconos.
- `WebP` para mockups.
- `PNG` transparencia o mayor nitidez.

## Peso recomendado

- Ideal: menor a 500 KB por imagen.
- Maximo sugerido: 1 MB por imagen para no hacer pesado el repo.
