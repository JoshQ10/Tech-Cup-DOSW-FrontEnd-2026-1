## Descripción

TechCup Fútbol es una aplicación web para la gestión de torneos con experiencia diferenciada por roles: Jugador, Capitán, Administrador, Árbitro y Organizador.

## Tecnologías

- React
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- ESLint

## Estructura del proyecto

```
src/
├── assets/       # imágenes, íconos, fuentes
├── components/   # componentes reutilizables
├── features/     # módulos por funcionalidad
├── pages/        # pantallas principales
├── hooks/        # custom hooks
├── services/     # consumo de APIs
├── store/        # manejo de estado
├── types/        # tipados globales
├── utils/        # helpers
├── routes/       # configuración de rutas
└── styles/       # estilos globales o tema
```
