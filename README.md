# TechCup Fútbol — Frontend

Frontend oficial de **TechCup Fútbol**, la plataforma digital para la gestión del torneo semestral de fútbol del Programa de **Ingeniería de Sistemas, Ingeniería de Inteligencia Artificial, Ingeniería de Ciberseguridad e Ingeniería Estadística** de la **Escuela Colombiana de Ingeniería Julio Garavito**.

Este frontend permite a jugadores, capitanes, árbitros, organizadores y administradores interactuar con la plataforma de forma intuitiva y organizada.

---

# Descripción

El **Frontend de TechCup Fútbol** es una aplicación web desarrollada con **React y TypeScript** que consume la API REST del backend para gestionar todas las funcionalidades del torneo.

La interfaz está diseñada para facilitar:

- Registro y autenticación de usuarios
- Gestión de equipos
- Organización de torneos
- Administración de partidos
- Visualización de resultados y estadísticas

# Objetivo del Frontend

El objetivo del frontend es proporcionar una **experiencia de usuario clara, rápida y organizada**, permitiendo que todos los participantes del torneo puedan interactuar fácilmente con la plataforma.

Se busca:

- Mejorar la organización del torneo
- Reducir procesos manuales
- Centralizar la información del campeonato
- Facilitar la interacción entre jugadores, capitanes, árbitros y organizadores
